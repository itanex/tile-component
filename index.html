Forward

The following page demonstrates the creation of a reusable component with a minimalistic approach to the html and css. The idea is to be able to use new components while leveraging the structure of a framework.

This tile does not leverage any of the bootstrap styling and only has a specific need to correct the font family because it is coming from a CDN and not a project configured scss compiled version.
